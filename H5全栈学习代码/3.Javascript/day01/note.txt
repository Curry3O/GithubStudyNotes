关键字：var if else for while continue
保留字：class static super int abstract
声明变量：var a;
初始化变量：a = 5;
声明并初始化变量：var a = 5;

命名规则：
由字母,数字，$和_ 组成；
不能以_和数字开头；
小驼峰命名法：userName;
不能使用关键字和保留字。
